Typography v1.1

Pinkspink Design System · Раздел типографики · 6 токенов размера, 3 веса, шрифт Ubuntu Mono

Спецификация типографики для аналитических дашбордов. Каждый токен покрывает несколько UI-ролей — благодаря этому шкала остаётся компактной (всего 6 размеров), но иерархия чётко выражена. Источник правды — tokens.json, текстовая версия — tokens.md, эта страница — визуальный референс.

2. Headings & Spacing v1.1

Универсальные правила отступов вокруг заголовков в нормальном flow. line-height: 1.25 на body. Margin-top заголовка считается как процент от его font-size, округление вниз до ближайшего чётного.

ТокенЗначениеФормула
--mt-h244px250% × 18 = 45 → 44 (round even down)
--mt-h324px175% × 14 = 24.5 → 24
--mt-h416px125% × 13 = 16.25 → 16
--tight-pair4pxЗазор заголовок ↔ описание
--content-gap8pxЗазор .meta ↔ контент

3. Графики — иерархия размеров

На графиках работают три размера сразу: datalabels (значения над столбиками) — на --fs-caption (11px), оси (X/Y tick labels) и легенда — на --fs-chart-meta (10px). Это намеренно: сами цифры читаются хорошо, оси и легенда уходят в фон и не «съедают» пространство.

4. Веса — три уровня

Используются три толщины начертания. Применяются на разных размерах в зависимости от роли (см. список ролей у каждого токена выше).

5. Letter-spacing

Один токен — --ls-label · 0.5px — применяется к UPPERCASE-лейблам, чтобы буквы «дышали» и не слипались.

С letter-spacing (правильно) сессии · конверсия · доход канал · страна · период

7. Фикс шрифта на формах

Form-элементы (<button>, <input>, <select>, <textarea>) не наследуют font-family от body в большинстве браузеров — применяется системный sans-serif. На моноширинном дашборде это смотрится чужеродно: всё вокруг — Ubuntu Mono, а кнопки — Helvetica.

Без фикса (баг)
↑ system sans-serif (Helvetica/Arial)
С фиксом
↑ Ubuntu Mono — наследуется от body

Фикс — одна строка CSS:

* { font-family: inherit; }

Универсальный селектор заставляет ВСЕ элементы наследовать font-family от родителя. body выставляет Ubuntu Mono → каскадно применяется ко всему, включая form-элементы.

8. Применение в новом проекте

Чтобы развернуть эту шкалу в любом другом дашборде — три шага:

1Подключить шрифт в <head>
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Ubuntu+Mono:wght@400;700&display=swap" rel="stylesheet">
2Скопировать :root блок в <style>
:root {
    --ff-mono: 'Ubuntu Mono', monospace;
    --fw-regular:  400;
    --fw-semibold: 600;
    --fw-bold:     700;
    --ls-label: 0.5px;
    --fs-kpi:        28px;
    --fs-h2:         18px;
    --fs-h3:         14px;
    --fs-body:       13px;
    --fs-caption:    11px;
    --fs-chart-meta: 10px;

    /* v1.1: heading spacing */
    --mt-h2: 44px;        /* 250% × 18 = 45 → 44 */
    --mt-h3: 24px;        /* 175% × 14 = 24.5 → 24 */
    --mt-h4: 16px;        /* 125% × 13 = 16.25 → 16 */
    --tight-pair: 4px;
    --content-gap: 8px;
}
* { font-family: inherit; }
body { font-family: var(--ff-mono); line-height: 1.25; }

h2 { font-size: var(--fs-h2); font-weight: var(--fw-semibold); margin: var(--mt-h2) 0 0; line-height: 1.25; }
h3 { font-size: var(--fs-h3); font-weight: var(--fw-semibold); margin: var(--mt-h3) 0 0; line-height: 1.25; }
h4 { font-size: var(--fs-body); font-weight: var(--fw-semibold); margin: var(--mt-h4) 0 0; color: var(--tx-secondary); line-height: 1.25; }

.meta { font-size: var(--fs-caption); color: var(--tx-secondary); line-height: 1.25; margin: 0 0 var(--content-gap); }
h2 + .meta, h3 + .meta, h4 + .meta { margin-top: var(--tight-pair); }
3Применять через var(--...)
h2 { font-size: var(--fs-h2); font-weight: var(--fw-semibold); }
.kpi-value { font-size: var(--fs-kpi); font-weight: var(--fw-bold); }
.kpi-label { font-size: var(--fs-caption); text-transform: uppercase; letter-spacing: var(--ls-label); }
table th { font-size: var(--fs-caption); font-weight: var(--fw-semibold); }
table td { font-size: var(--fs-caption); }
4Для Chart.js — поднять глобальные defaults
Chart.defaults.font.family = "'Ubuntu Mono', monospace";
Chart.defaults.font.size = 10;  // = --fs-chart-meta (оси и легенда)
Chart.defaults.plugins.datalabels = Chart.defaults.plugins.datalabels || {};
Chart.defaults.plugins.datalabels.font = { size: 11 };  // = --fs-caption
