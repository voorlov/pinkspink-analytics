Controls v1.0

Pinkspink Design System · Кнопки, фильтры, чекбоксы, навигация · 4 компонента · 28px / 2px · 5 состояний

Спецификация контролов. Filter-кнопки и квадратные переключатели — на одной высоте 28px (стоят пиксель-в-пиксель). Top-level навигация — отдельный компонент .tab с подчёркиванием, без заливки. Чекбоксы — кастомные с чёрной заливкой при checked. Полное соответствие Typography v1.0. Источник правды — tokens.json.

1. Базовые токены

ТокенЗначениеЧто это
--h-control28pxВысота всех контролов (.btn, .btn-sq). Square кнопки = квадрат 28×28.
--r-control2pxРадиус всех контролов. Меньше чем у карточек/таблиц — даёт «инженерный» вид.
 
:root {
    /* Controls v1.0 — дополнение к Typography v1.0 */
    --h-control: 28px;
    --r-control: 2px;
}

3. Filter-bar и filter-group

Когда несколько filter-кнопок объединены в смысловую группу — оборачиваются в .filter-group с UPPERCASE-подписью + 1px полоской под ней. Несколько групп идут side-by-side в .filter-bar.

Подпись группы — --fs-caption 11px regular UPPERCASE с --ls-label letter-spacing, цвет --tx-secondary. Тонкая полоска снизу — 1px solid --bg-border. Min-width 110px чтобы группы выравнивались.

5. Принципы

  • Одна высота — все контролы на --h-control 28px. В filter-баре они стоят пиксель-в-пиксель.
  • Один радиус — все контролы на --r-control 2px. Карточки/таблицы — свои радиусы (4 / 6 / 8).
  • Active = тёмная заливка для .btn и .btn-sq. Чёрный фон + белый текст.
  • Навигация — особый случай — .tab с подчёркиванием, без заливки. Визуально отличается от filter-кнопок.
  • Focus всегда виден — outline 2px бренд-фиолетовый с offset 2px. Не убирать outline: none без замены.
  • Никаких хардкод-размеров — всё из Typography v1.0.

6. Применение в новом проекте

1Применить Typography v1.0

См. соседнюю папку typography/. Без неё контролы не будут работать как задумано (шрифт, размеры).

2Добавить токены контролов в :root
:root {
    --h-control: 28px;
    --r-control: 2px;
}
3Скопировать CSS компонентов

Из tokens.md → секция «Четыре компонента». Готовые блоки для .btn, .btn-sq, .tab, .check и контейнеров.

4Использовать классы
<!-- Filter-bar с двумя группами -->
<div class="filter-bar">
    <div class="filter-group">
        <div class="filter-label">GRAIN</div>
        <div class="filter-buttons">
            <button class="btn-sq">D</button>
            <button class="btn-sq is-active">W</button>
            <button class="btn-sq">M</button>
        </div>
    </div>
</div>

<!-- Top-level навигация -->
<div class="tab-bar">
    <button class="tab">Сводка</button>
    <button class="tab is-active">Воронки</button>
    <button class="tab">Аналитика</button>
</div>

<!-- Чекбоксы -->
<label class="check"><input type="checkbox" checked> ig</label>
<label class="check"><input type="checkbox"> google</label>
