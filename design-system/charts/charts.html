Charts v1.0

Pinkspink Design System · Раздел charts · 6 типов · 4 форматтера · 4 пресета легенды · 4 ступени высоты · единый hover

Спецификация графиков. Всё работает на Chart.js 4.4 + chartjs-plugin-datalabels 2.2. Полностью наследует типографику из Typography v1.0 — никаких новых размеров шрифта или весов вне согласованной шкалы. Источник правды — tokens.json, текстовая версия — tokens.md.

1. Глобальная настройка

Запускается один раз при загрузке страницы. Дальше все графики наследуют шрифт, цвет осей, размеры подписей, hover-mode.

// === Chart.js global setup ===
Chart.register(ChartDataLabels);  // ОБЯЗАТЕЛЬНО для v4 — иначе datalabels игнорируются

Chart.defaults.font.family = "'Ubuntu Mono', monospace";
Chart.defaults.color       = '#636E72';                 // --tx-secondary

Chart.defaults.font.size                  = 10;            // --fs-chart-meta (оси)
Chart.defaults.plugins.legend.labels.font = { size: 10 };  // --fs-chart-meta (легенда)
Chart.defaults.plugins.datalabels         = Chart.defaults.plugins.datalabels || {};
Chart.defaults.plugins.datalabels.font    = { size: 11 };  // --fs-caption (datalabels)

Chart.defaults.scale.grid.color = 'rgba(0,0,0,0.06)';

Chart.defaults.interaction = { mode: 'index', intersect: false };
// (для bubble override на { mode: 'point', intersect: true } per-chart)

2. Форматтеры

Четыре функции, единые для всех графиков. Все автоматически скрывают 0 и null. Локаль ru-RU для разделителя тысяч.

const fmt = {
    num: (v)         => (v == null || v === 0) ? '' : v.toLocaleString('ru-RU'),    // 12 845
    pct: (v, dec=1)  => (v == null || v === 0) ? '' : v.toFixed(dec) + '%',         // 12.4%
    sec: (v)         => (v == null || v === 0) ? '' : v + 's',                      // 31s
    cur: (v, c='$')  => (v == null || v === 0) ? '' : c + v.toLocaleString('ru-RU') // $630
};
Функция01234512.456null
fmt.num ''12 34512.456''
fmt.pct ''12345.0%12.5%''
fmt.sec ''12345s12s''
fmt.cur ''$12 345$12''

3. Datalabel-пэд

Полупрозрачный фон под значениями — чтобы число читалось когда падает на цветную линию или пузырь. Применяется к Line, Combo (line-часть), Bubble. Для bar-сегментов не нужен (там белое на цветном).

const dlPad = {
    backgroundColor: 'rgba(255,255,255,0.85)',
    borderRadius:    3,
    padding: { top: 1, bottom: 1, left: 4, right: 4 }
};

6. Высоты — четыре ступени

Каждый график получает min-height и max-height равные одной из четырёх CSS-переменных. Контейнер фиксированный — график не «дышит» при разных данных.

:root {
    --ch-xs: 60px;   /* sparkline в KPI-карточках */
    --ch-sm: 180px;  /* графики в слайдер-карточках (компакт) */
    --ch-md: 280px;  /* стандартный график в .cell */
    --ch-lg: 360px;  /* детализированные с осями + большая легенда */
}
--ch-xs · 60px
--ch-sm · 180px
--ch-md · 280px
--ch-lg · 360px

7. Ширина баров — maxBarThickness

На каждом dataset с type: 'bar' ставим maxBarThickness: 40. Бары не превышают 40px, после порога — зазоры. Когда баров слишком много (например, 30 дней) — переключатель grain (day/week/month) вместо горизонтального скролла. Скролл — fallback только когда агрегация невозможна.

{ type: 'bar', ..., maxBarThickness: 40, categoryPercentage: 0.8, barPercentage: 0.9 }

8. Цветовая логика

Все цвета на графиках берутся только из токенов (Color v1.0 когда будет создан, или Python-словарей CHANNEL_COLORS / FUNNEL_STAGES). Никаких inline hex.

  • channel — Социальные: Social, Paid, Direct, Organic, Referral, Email
  • funnel — Этапы воронки: home, catalog, product, ATC, checkout, purchase
  • semantic — Дельты, sparkline по тренду, акценты: growth, decline, neutral, highlight

9. Соответствие Typography v1.0

Никаких новых размеров шрифта или весов вне согласованной шкалы. Все настройки идут через Chart.defaults:

  • Datalabels (значения, source-имена на пузырях): --fs-caption 11px / regular
  • Подписи осей и легенды: --fs-chart-meta 10px / regular
  • Sparkline min/max: --fs-chart-meta 10px / regular
  • Шрифт: 'Ubuntu Mono' через Chart.defaults.font.family

Никаких weight: 'bold' в чарт-конфигах. Если хочется выделить — используем цвет (--c-growth, --tx-ondark, --c-decline), не толщину.
