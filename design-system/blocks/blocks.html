Blocks v1.1

Pinkspink Design System · Блоки-карточки · 2026-04-29

Спецификация блоков. Один класс .block заменяет цепочку .h4x6 + .title-area + .cell. Высота — auto по контенту, ритм через единые chart-heights. v1.1: добавлен принцип «карточка не вкладывается в карточку» — wrappers внутри блока (.chart-wrap, .data-table-wrap) naked. Полное соответствие Typography v1.1, Charts v1.0, Tables v1.1. Источник правды — tokens.json.

3. Height-классы (для chart-блоков)

Применяют --ch-* токены из Charts v1.0 к chart-wrap внутри блока. Visual ритм между блоками сохраняется через одинаковые chart-heights.

КлассВысота chart-wrapПрименение
.h-xs--ch-xs 60pxSparkline в KPI-карточках
.h-sm--ch-sm 180pxСлайдер-карточки, компактные графики
.h-md--ch-md 280pxDefault chart (большинство случаев)
.h-lg--ch-lg 360pxBubble-чарты, dual-axis, детализированные

4. Анатомия — структура и CSS

Стандартная HTML-разметка одного блока:

<div class="grid">
    <div class="block span-6 h-md">
        <h3>Заголовок блока</h3>
        <p class="meta">Описание</p>
        <div class="chart-wrap">
            <canvas>...</canvas>
        </div>
    </div>
</div>

Полный CSS:

.grid {
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: var(--sp-4);
    /* НЕТ grid-auto-rows — ряды auto по контенту */
}

.block {
    background:      var(--bg-card);
    border-radius:   var(--r-xl);
    box-shadow:      var(--sh-card);
    padding:         var(--sp-3);
    display:         flex;
    flex-direction:  column;
}

/* Первый child не получает свой margin-top */
.block > :first-child { margin-top: 0; }

/* Span-классы */
.span-12 { grid-column: span 12; }
.span-8  { grid-column: span 8; }
.span-6  { grid-column: span 6; }
.span-4  { grid-column: span 4; }
.span-3  { grid-column: span 3; }
.span-2  { grid-column: span 2; }

/* Height-варианты */
.block.h-xs .chart-wrap { height: var(--ch-xs); }   /* 60px */
.block.h-sm .chart-wrap { height: var(--ch-sm); }   /* 180px */
.block.h-md .chart-wrap { height: var(--ch-md); }   /* 280px */
.block.h-lg .chart-wrap { height: var(--ch-lg); }   /* 360px */

6. Принципы

  • Один класс = один блок — никакой цепочки .h4x6 + .title-area + .cell
  • Высота auto по контенту — нет фиксированных grid-row span
  • Визуальный ритм через chart-heights (--ch-md у одного типа = одинаковая высота блоков)
  • Заголовки внутри блока — глобальные правила Typography v1.1, никаких font-overrides
  • Span-классы — только ширина, высота отдельно через .h-*
  • Padding 12px (--sp-3) из spacing-шкалы (заменяет хардкод 14px)
