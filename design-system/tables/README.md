# Tables — v1.1

Раздел дизайн-системы про таблицы. Auto-fit высота по контенту до 20 строк, дальше — внутренний scroll со sticky-header. Высота строки = высоте контролов (28px) — единый ритм через всю UI. **С v1.1** `.data-table-wrap` — naked: фон/скругление/тень приходят от `.block`-обёртки.

**Зависит от:** [Typography v1.1](../typography/), [Controls v1.0](../controls/), [Blocks v1.1](../blocks/).

## Базовые токены

| Токен | Значение | Что это |
|-------|----------|---------|
| `--row-h-table` | `28px` (= `--h-control`) | Высота строки = высоте контролов |
| `--max-rows-table` | `20` | Максимум видимых строк до scroll |
| `--max-h-table` | `588px` (= 21 × 28) | Вычисляемая max-height обёртки |

## Поведение

| Случай | Высота | Scroll | Sticky header |
|--------|--------|--------|---------------|
| **1-20 строк** (короткая) | По контенту | Нет | Не нужен |
| **>20 строк** (длинная) | 588px фикс | Да | Активен |

## Структура

```html
<div class="block span-12">
    <div class="data-table-wrap">
        <table class="data-table">
            <thead><tr><th>...</th></tr></thead>
            <tbody><tr><td>...</td></tr></tbody>
        </table>
    </div>
</div>
```

`.block` — карточка (фон, скругление, тень). `.data-table-wrap` — naked: только `max-height` и `overflow-y`. `.data-table` — сама таблица без обёрточных стилей.

## Принципы

1. **Auto-fit высота** — таблица сама определяет размер контейнера до 20 строк
2. **Внутренний scroll** при превышении — заголовки sticky, остаются видимыми
3. **Компактность по дефолту** — никаких opt-in modifier'ов (`.tbl-compact` упразднён)
4. **Высота строки 28px = высоте контролов** — через всю UI один ритм
5. **Тонкий scrollbar 6px** — не съедает место, не доминирует
6. **Никаких grid-cell-обёрток** с фиксированной высотой — обычный flow
7. **`.data-table-wrap` — naked** (v1.1) — только функциональные свойства; карточка приходит от `.block`

## Что упраздняется при миграции дашборда

- **`.tbl-compact` modifier** — компактность теперь дефолт, класс убирается из HTML и CSS
- **`.h8x12` grid-cell для таблиц** — заменяется на обычный flow `h2/h3 + .meta + .data-table-wrap`

## Соответствие другим разделам

- **Typography v1.1** ✓ — th/td на `--fs-caption` 11px, lh 1.25
- **Controls v1.0** ✓ — `--row-h-table` = `--h-control` = 28px (фильтр-кнопки и строки таблицы пиксель-в-пиксель)
- **Charts v1.0** ✓ — не пересекаются (разные контейнеры)
