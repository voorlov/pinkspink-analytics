13. KPI-карточки — три варианта

В дашборде используются три формата KPI: компактная 2×2 без графика, 2×2 со спарклайном внизу, и блок-сетка .kpi-grid на странице «Сводка» (карточки разной ширины/высоты). Цифры в них — это --fs-kpi-xl (28px) или --fs-kpi-lg (20px) в зависимости от плотности.

14. Графики — пять типов

В дашборде используется 5 паттернов Chart.js. У каждого свои конвенции по подписям (datalabels), легенде и формату значений. Шрифт, цвет осей и сетки берутся из TOKENS["chart"] через Chart.defaults (применяется один раз при загрузке).

A — Grouped Bar (воронка по периодам)

Где: блок «Динамика воронок». Подписи: anchor:'end', align:'top' — над столбиком, формат v > 0 ? v : '' (нули прячем). Легенда: сверху. Шкала Y: beginAtZero:true, grace:'15%' для воздуха над столбиками.

B — Line с подписями (конверсия %)

Где: «catalog→product», «product→ATC» и т.п. в воронке. Подписи: align:'top', формат v.toFixed(1) + '%'. Tooltip: custom callback с %. Шкала Y: ticks с v + '%'.

C — Stacked Bar + Line (комбо, dual-axis)

Где: Сессии по каналам (stacked) + Покупатели (линия) на одном графике. Подписи: display:false для bar, или внутри столбиков (color:#FFF). Оси: Y слева для bar, Y1 справа для line, grid.drawOnChartArea:false у Y1.

D — Bubble (эффективность source)

Где: «Каталог→Товар», «Товар→Корзина» — соотношение объём × конверсия × размер. Оси: X = сессии на этапе, Y = конверсия в следующий, размер пузыря = объём. Подписи: по умолчанию выключены, label виден в tooltip.

E — Sparkline (мини-линия в KPI)

Где: внутри .kpi-spark в KPI-карточках. Подписи и легенда: display:false. Оси: скрыты (display:false). Только линия и точки. Нужна для тренда «глазом», не для считывания значений.
